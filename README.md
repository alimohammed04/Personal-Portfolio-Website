# Ali Mohammed Portfolio

A responsive personal portfolio website built with **HTML**, **CSS**, and **JavaScript** to showcase my projects, technical skills, and career journey.

## Overview

This portfolio was created to present my work, highlight my technical background, and reflect my growth as a frontend developer and aspiring .NET full stack developer.

The website includes a clean and modern interface with responsive layouts, project cards, dark mode support, and dynamic project rendering from a JSON file.

## Features

* Responsive design for desktop, tablet, and mobile
* Dark and light mode toggle
* Projects section with category filtering
* Dynamic project rendering using JSON data
* About section
* Skills section
* Career journey section
* Accessible buttons, links, and headings
* Organized and maintainable file structure

## Technologies Used

## Getting Started

To run the project locally:

1. Clone the repository:

```bash
git clone git@github.com:alimohammed04/Personal-Portfolio-Website.git
```

2. Open the project folder:

```bash
cd Personal-Portfolio-Website
```

3. Open `index.html` in your browser

You can also use Live Server in VS Code for easier development.

## Projects Data

The projects section is loaded dynamically from:

```bash
data/projects.json
```

This makes it easier to update project cards without editing the HTML manually.
